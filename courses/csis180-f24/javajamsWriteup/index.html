JavaJam Case Study

Background

You want to help a friend develop a web presence for their shop that will display the services and provide a calendar for the performances. They would like a home page, menu page, music performance schedule page, and a job opportunities page.

Part 1

The JavaJams project has some of the pages written, but you'll notice they aren't very exciting and look unprofessional. Throughout this case study we will make changes to make it look better.

Let's start:

  • Create an external style sheet named javajam.css that configures the color and text for the JavaJam website.
  • Since we are going to be using CSS, remove any references to style found in the starting index.html (i.e. strong, em, small)
  • Using the image below and color values, modify the Home page to look like the image. Make sure you real all information in Part 1 before starting.
    #221811
    #FCEBB6
    #D2B48C
    #8C3826
    #FEFAEB
  • Some things you may not notice and/or information you may need:
    • header and nav bar are centered
    • nav items are bold
    • h1 element is 200% line height
    • footer has small font size, italics, and centered text.
    • A div with an id of wrapper needs to be created to contain all the contents in the body section.
    • This wrapper contains the contents of the page. The wrapper is centered on the overall page. Be sure to open it in a browser to ensure that it does indeed center overall.
  • Once your index.html page looks like the example, make the necessary changes to menu.html so that it has the same appearance.

Part 2

For this part we are going to work towards making changes to the home page so it looks like this image: Again, make sure you read through the steps first.

    wrapper
    header
    nav
    div with large image
    main

  2. Replace the "Relax at JavaJam" text contained within the h2 element with "Follow the Winding Road to JavaJam"
  3. Add a paragraph with the following text below the h2 element and above the existing paragraph:
    We're a little out of the way, but take a drive in the country down Garrett Bay Road to JavaJam today! Indulge in our locally roasted free-trade coffee and home-made pastries. You'll feel right at home at JavaJam!
  4. Configure the text "JavaJam Coffee Bar Features:" within an h3 element below the second paragraph and above the unordered list.
  5. Modify the CSS to display the winding road image as the background image. It should not repeat, it should fill the whole window, it should be fixed and the background size should be cover.
  6. Add a new selector for an id named homehero to configure 300px height and to display the hero.jpg background image to fill the space without repeating.
  7. Modify the CSS so that the wrapper id has a minimum width of 900px and maximum width of 1280px. It should also add a drop-shadow effect on the wrapper.
  8. Add CSS to set top and bottom padding of header to 5px.
  9. Modify the CSS so the h1 element uses a font size of 3em.
  10. Modify the nav CSS to use a font size of 1.5em, 5px of top padding, and 5px of bottom padding.
  11. Add CSS to prevent the hyperlinks in the nav area from displaying the default underline.
  12. Modify the CSS so that the footer has 1em of padding and a 2px top border using the same color as the headings.
  13. Add CSS so that the main element has 2em of padding on the left, right, and bottom.

Part 4

Now we will create the music page. The final result should look like this image:
  1. Copy the menu.html file and rename it music.html.
  2. Change the page title to an appropriate phrase.
  3. Delete the image and description list from the page.
  4. Replace the text in the h2 to be "Music at JavaJam"
  5. Add a paragraph with the following text below the h2 element:
    The first Friday night each month at JavaJam is a special night. Join us from 8pm to 11pm for some music you won't want to miss!
  6. The remaining content of the page will consist of two areas describing music performances. The area describing each music performance consists of an h4 element, a div of the class details, and an image link.
  7. The h4 elements should be "January" and "February"
  8. After each of the h4 elements should be a div of class details containing both the image and the text.
  9. Each image is a thumbnail that links to the full size image.
  10. The text for the two sections are:
    • Melanie Morris entertains with her melodic fold style.
    • Tahoe Greg is back from his tour. New songs. New stories.
  11. Continue adding styling to CSS until your page looks like the image above.
